# Quick Start Guide

## Getting Started in 3 Steps

### 1. Open the Application
```bash
cd an_amazing_ton/app
# Just open index.html in your browser!
```

Or double-click: `/home/user/an_amazing_ton/app/index.html`

### 2. Upload Your Training File
- Click "Upload Training File" button
- Select `training.xlsx` from the project root
- Wait for parsing (takes ~10ms)
- Your trainings appear!

### 3. Start Using
- **Browse:** Navigate weeks in sidebar
- **Search:** Filter by exercises, intensity, set type
- **Timer:** Run Tabata timer with your workouts
- **Export:** Save trainings as PDF for gym use
